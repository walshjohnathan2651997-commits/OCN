# CESE-OCN Paper Version Index

## V1_original_OCN

This folder preserves the original CESE-OCN manuscript family. The key source file is:

- `V1_CESE_OCN_submit_ready_controlled_diagnostic.docx`

V1 is useful as the conceptual blueprint. It contains the stronger shared-threshold ordinal calibration story, CESE-OCN architecture, and broad diagnostic framing. It should not be treated as the current empirical manuscript because several method claims are stronger than the available pilot evidence.

## V2_evidence_aware_hierarchical

This folder contains the revised evidence-aware hierarchical pilot manuscript. The key current file is:

- `V2_CESE_OCN_evidence_aware_hierarchical_pilot.docx`

V2 keeps the CESE-OCN problem framing but moves the empirical mainline to escalation_binary, frozen NLI cross-encoder evidence-specific signal, and strict silver-label positioning. It remains useful as the evidence-aware hierarchical predecessor to V3, but it is no longer the recommended working manuscript.

## V3_mixed_framework_v2_R4

This folder contains the current mixed-framework rewrite based on the frozen R4 conservative strong router. The key current file is:

- `CESE_OCN_mixed_framework_v2_R4_draft.docx`

V3 is the recommended working manuscript. It reframes CESE-OCN as a mixed evidence-relation calibration framework rather than a single flat classifier. The current empirical mainline is the frozen `R4_conservative_strong` router: contradiction first, conservative strong-action routing, and supported/mild residual classification. The strongest writable result is that R4 improves strong-action positive-F1 from 0.2408 to 0.3967 while preserving contradiction and escalation performance on silver-label pilot data.

## Current recommendation

Use V3 for all future paper editing. Keep V1 as historical/theoretical reference and V2 as the evidence-aware hierarchical predecessor.
